# TURKA Araç Muayene Dashboard

Araç muayene istasyonları için hazırlanmış demo dashboard.

## Özellikler

- Genel bakış ve canlı ağ akışı
- İstanbul Anadolu / Avrupa istasyon haritaları
- EV güvenlik analizi
- Müşteri analizi
- Operasyon simülasyonu
- TR/EN dil desteği

## Güvenlik Notu

Bu GitHub sürümünde gerçek API anahtarı veya hassas bilgi bulunmaz. AI analizleri demo/yerel analiz mantığıyla çalışır.
